# Capsite — Коллекция шаблонов оформления для узлов

Данный репозиторий содержит библиотеку визуальных оформлений (лендингов), предназначенных для использования с установочным скриптом [remnascrypt](https://github.com/imdeist/remnascrypt). 

Все доступные варианты хранятся в директории `/cups`. При установке узла скрипт автоматически сканирует содержимое этой папки и предлагает пользователю выбрать понравившийся дизайн.
