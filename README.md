# 🌌 Capsite

[![Integration](https://img.shields.io/badge/integration-remnascrypt-blue.svg?style=flat-square)](https://github.com/imdeist/remnascrypt)
[![UI](https://img.shields.io/badge/UI-Templates-success.svg?style=flat-square)](#)

> **Коллекция элегантных презентационных веб-интерфейсов для ваших узлов.**

**Capsite** — это специализированная библиотека современных визуальных решений (лендингов) и стартовых страниц. Данный репозиторий выступает в роли единой базы дизайнов, созданной для бесшовной работы в связке с автоматизированным установочным скриптом [remnascrypt](https://github.com/imdeist/remnascrypt).

---

### ✨ Ключевые особенности

* **Готовые решения:** Набор профессионально сверстанных страниц для моментального развертывания на ваших серверах.
* **Бесшовная интеграция:** Скрипт `remnascrypt` автоматически сканирует директорию `/cups` и предлагает выбрать стилистику прямо в процессе установки узла.
* **Легкость кастомизации:** Любой шаблон имеет чистый код и легко адаптируется под индивидуальные задачи или брендинг.
* **Простота расширения:** Архитектура репозитория позволяет легко и быстро добавлять собственные варианты оформления.

---

### 🚀 Как это работает

Вся настройка визуальной части происходит автоматически. Вам не нужно вручную копировать и редактировать файлы для каждого нового узла:

1. Запустите скрипт [remnascrypt](https://github.com/imdeist/remnascrypt) для инициализации и установки узла.
2. Скрипт самостоятельно обратится к репозиторию **Capsite** и проанализирует содержимое директории `/cups`.
3. В терминале появится интерактивное меню с предложением выбрать дизайн из списка доступных.
4. Выбранный презентационный интерфейс будет автоматически скачан и развернут на вашем узле.

---

### 📂 Структура репозитория

Все доступные шаблоны бережно отсортированы и хранятся в директории `/cups`. Каждый отдельный дизайн находится в собственной изолированной подпапке, что делает навигацию интуитивно понятной.
